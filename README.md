# GCP-AI-Platform-Trainjob
Training a model on GCP AI Platform

## Objective & Idea
This is not about modeling, but training a Keras NN on gcloud AI Platform.

## Prerequisites
1) Setup a venv
2) `pip install -r requirements.txt`
3) Store GCP User keys as `keys.json`
